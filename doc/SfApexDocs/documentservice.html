DocumentService

public with sharing class DocumentService
DescriptionThis class is the entry point for DocuSign for Salesforce envelope document management. It provides methods to retrieve Salesforce source documents and add or remove documents from an envelope.

  getDocumentFolders

public static Document.Folders getDocumentFolders(final SObjectType type)
DescriptionRetrieves a list of Salesforce source documents folders by object type. The current user's personal folder, and the documents contained within, is always returned as the first element in the list. The returned folder IDs may be used in subsequent calls to DocumentService.getFolderDocuments.

By default, the Salesforce Document and ContentVersion types are supported. More types may be supported via plugins.

The caller must be assigned the DocuSign User permission set.

ReturnsA list of document folders.
Parameters
typeThe Salesforce object type.
SeeDocumentService.getFolderDocuments

  getLinkedDocuments

public static List<Document> getLinkedDocuments(final SObjectType type, final Set<Id> linkedEntityIds, final Boolean getLastUpdatedOnly)
DescriptionRetrieves a list of Salesforce source documents by linked entity IDs.

For example, to pull in all of an Opportunity's Notes and Attachments related list: 

 List docs = DocumentService.getLinkedDocuments('ContentVersion', new Set { opp.Id }, false);

By default, only ContentVersion types are supported. More types may be supported via plugins.

The caller must be assigned the DocuSign User permission set.

ReturnsA list of DocuSign envelope documents.
Parameters
typeThe Salesforce object type.
linkedEntityIdsThe Salesforce parent IDs.
getLastUpdatedOnlyWhether to retrieve only the last updated linked document.
