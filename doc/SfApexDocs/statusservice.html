StatusService

public with sharing class StatusService
DescriptionThis class is the entry point for DocuSign for Salesforce envelope and recipient status management. It provides methods to retrieve and take action upon DocuSign for Salesforce statuses.

Methods

  createStatus

public static List<Envelope.Status> createStatus(final List<Envelope.Status> envelopes)
DescriptionCreates envelope status records from a list of sent envelopes. This will fail if any of the envelopes already have a status record associated with it.

The caller must be the owner of the envelopes.

ReturnsThe new envelope statuses.
Parameters
envelopesThe DocuSign envelopes from which to create statuses. These must have been previously sent with a non-null docuSignId.
