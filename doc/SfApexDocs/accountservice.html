LoginStatus
public enum LoginStatus
Login statuses:
  • Success: Login was successful and mapped to a single valid DocuSign account.
  • Failure: Invalid DocuSign username or password.
  • SelectAccount: Login was successful, but maps to mulitple valid DocuSign accounts.

  createTrial

public static Account.Membership createTrial(final Account.Trial trial)
DescriptionCreates a DocuSign for Salesforce trial account. This account may be prepared via AccountService.prepareTrial.

Trial accounts are validated to ensure:

  • The trial account is non-null
  • The trial email is a valid email address
  • The trial user's name is not blank
  • The trial company is not blank
  • The trial user has agreed to DocuSign's terms and conditions.

The caller must be assigned the DocuSign Administrator permission set.

ReturnsDetails about the newly-created trial account membership.
Parameters
trialThe trial account to create.
SeeAccountService.prepareTrial for details on how the trial account can be prepared.

  login

public static LoginResult login(final String username, final String password, final Environment.Target target, final Url otherUrl, Long selectedAccountNumber)
DescriptionAttempts login to DocuSign with the supplied credentials. This must be invoked successfully prior to any other DocuSign for Salesforce actions. If the login is successful and maps to a single valid DocuSign account, DocuSign for Salesforce will be updated with these credentials. If the login is successful and maps to valid multiple accounts, those account options are returned to the caller to select the desired account in a subsequent call to this method.

The caller must be assigned the DocuSign Administrator permission set.

ReturnsThe result of the login. This may succeed with a single account (no additional action is required to configure DocuSign for Salesforce), or with multiple accounts in which case this method should be called again with the same parameters and selectedAccountNumber defined with the desired account.
Parameters
usernameThe DocuSign username.
passwordThe DocuSign password.
targetThe DocuSign environment. Valid values are Stage, Demo, Production, or Other. If Other is specified, a valid DocuSign base URL must be supplied via the otherUrl parameter.
otherUrlThe DocuSign base URL. This must be a valid DocuSign URL if target is Other
selectedAccountNumberThe DocuSign account number for the desired DocuSign for Salesforce account. Used to select an account when the user credentials map to multiple valid DocuSign accounts.
