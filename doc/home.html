DocuSign for Salesforce Apex Toolkit

These pages document the Apex code for the DocuSign for Salesforce Apex Toolkit. This is a collection of Apex types and utilities that allow developers to administer DocuSign for Salesforce, send envelopes, and retrieve status.

The starting points for any integration are the service classes grouped by function:

  • AccountService: Configure DocuSign account settings.
  • EnvelopeService: Create and send DocuSign envelopes.
  • DocumentService: Manage DocuSign envelope documents.
  • RecipientService: Manage DocuSign envelope recipients.
  • TemplateService: Manage DocuSign templates.
  • SigningService: Methods to support DocuSign-ing.
  • StatusService: Methods supporting status-related actions: view, correct, resend, and void.
